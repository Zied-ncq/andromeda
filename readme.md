# Bpmn Engine for microservice cloud era.

> A Node.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```
## Prerequisites
- Nodejs
- MongoDB

`.env file`
``` properties
MONGODB_URI=mongodb://127.0.0.1:27017/andromeda
#ACTIVE_MODULES=server, persistence
ACTIVE_MODULES=server

```
